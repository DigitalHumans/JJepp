그외 트리비아
1. 리눅스에서 윈도우 exe 파일을 만들려면 리눅스에 mingw설치후 
x86_64-w64-mingw32-g++ (컴파일 할 파일 이름) -std=c++17
(단, 그냥 vs에서 clang으로 컴파일 하는것이 더 편함)
2. 함수 파라미터에서 
int a: 복사후 이용. 해당 변수에 영향을 주지 않음. 제일 느림(revese 연산 한번정도)
int *a:포인터로 이용 .을 ->로 써야 하는등 문법을 고쳐써야 함. 해당 변수에 영향을 줌
int &a: 레퍼런스. 효율은 포인터와 비슷하거나 연산면에서는 더 적음.
4.mingw를 쓰던 g++을 쓰던, 윈도우로 옮기면 글씨가 깨진다
이럴때는 cmd에서 chcp 65001 을 입력하면 된다.
