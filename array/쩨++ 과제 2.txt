배열
연속된 같은 자료형의 변수를 하나로 묶어서 사용하는 것.
같은 자료형의 변수 여러 개를 쓰는 것보다는 배열 하나를 쓰는 게 더 편하다.

문자열
C는 char 형의 배열, 가장 뒤에 NULL문자가 붙음.
C++은 string과 char 형의 배열 둘 다 사용 가능하나 string 권장.

배열 선언
int array[10] = { 0 }; 10개짜리 정수 배열을 선언하고 모두 0으로 초기화.
초기화하지 않은 채로 배열 선언 시 이후에 각 원소별로 초기화해야 함.

배열의 크기를 동적으로 하는 방법 : 동적할당, vector
동적할당은 변수를 이용한 크기 지정 가능
vector는 프로그램 실행 도중에 크기가 유동적으로 바뀜.

표준 입출력
Standard Input Output
C언어에서는 stdio.h
C++에서는 iostream

표준 입출력 수행
<<, >>을 스트림 연산자라고 한다.
쉬프트 연산자와 같은 기호이지만 서로 다른 연산자.
<<은 아웃스트림 연산자, >>은 인스트림 연산자이다.

hello world 출력시 cout << "Hello World" << endl;
스트림 연산자는 방향을 반대로 하면 안 된다.

표준 출력
cout이다.
해당 자료형이 기본 자료형에 속할 경우 각 자료형에 맞게 알아서 출력해준다.
bool은 0 또는 1로 출력된다.
자료형을 섞어도 잘 작동한다.
cout << 13 << "Hello" << a << endl; 은 잘 작동한다. (a는 float로 선언 및 초기화)

표준 입력
cin이다.
받아야 할 자료형에 맞게 최대한 변환 후 각 변수에 넣어준다.
실패 시 cin.fail()은 참이 된다. cin.clear()로 실패 플래그를 초기화할 수 있다.
cin.ignore()로 특정 글자가 나오거나 특정 길이만큼 입력을 무시할 수 있다.
이 세 메서드를 이용해 에러 핸들링을 할 수 있다.
cin >> a >> b >> c; //a, b, c는 기본 자료형의 이미 선언된 변수들.
가급적 하나씩 받는 게 좋음.
